# Tube

Ferramenta CLI Simples para baixar audio do YouTube

## Dependências

- yt-dlp
- ffmpeg

## Dependẽncias Python

- inquirerpy

## Uso

```bash
python main.py
```

ou

```bash
python main.py URL
```

## Créditos

Esse projeto foi originalmente baseado em [Kafkarlos/Youtube-MP3-Converter](https://github.com/Kafkarlos/Youtube-MP3-Converter).

Essa versão adiciona melhorias de interface, usabilidade e organização do código.
